3. Shadow Rate Limiting (Anti-Abuse)

If the "Bouncer" (Layer 1) detects borderline malicious intent (e.g., subtle prompt injection attempts):

  • Latency Injection: Delay the response by 10-20 seconds to slow down automated scripts.
  • Fidelity Degradation: Return a simplified AI response that doesn't reveal internal system state.
  • Silent Logging: Alert mods in #bot-logs without notifying the user.
