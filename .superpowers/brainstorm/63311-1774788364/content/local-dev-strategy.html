The "One-Command" Environment

A single npm run dev starts the entire development stack:

  • Bot: nodemon + ts-node
  • Dashboard: Next.js Dev Server
  • Database: Supabase CLI (Docker)
  • Tunnel: ngrok / localtunnel
  • AI: Ollama (Local LLM)
  • Mail: Inbucket (for auth emails)

Why this is better

  • Zero Production Risk: Your experimental code never touches the 36k member server until it's merged.
  • Speed: Local Supabase and local AI mean instant feedback loops.
  • Consistency: Every developer on the project uses the exact same Docker-based environment.

Setup Requirement

  • Docker: Developers need Docker installed to run the local Supabase and Ollama containers.
