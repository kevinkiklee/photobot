Agnostic AI Strategy: Gemini & Local Testing

Ensuring high-performance production with Gemini and cost-free local development.

1. The AI Provider Interface

Unified LLM Wrapper

A single analyzeImage() function that doesn't care which model is running.

// config.ts
AI_PROVIDER: 'gemini' | 'ollama' | 'openai'

// provider.ts
if (AI_PROVIDER === 'gemini') return GeminiSDK;
if (AI_PROVIDER === 'ollama') return LocalOllama;

✓ Swap models in .env without touching a single line of feature code.

2. Development & Testing Flow

Local LLM (Ollama)

Test complex photography analysis for $0.

  • Ollama Integration: Use llava or moondream locally for vision testing.
  • Mock Provider: A "Dry Run" mode that returns static JSON for ultra-fast unit tests.

Gemini Optimization

We'll leverage Gemini-specific features while maintaining compatibility:

  • Gemini 1.5 Flash (Bouncer): Extremely fast and cheap for our Layer 1 security check.
  • Gemini 1.5 Pro (Analyzer): High-tier vision for Phase 3 "Reverse Engineering" and "Troubleshooting."
  • JSON Mode: Forcing Gemini to return structured data for the palette extractor.
